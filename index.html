Position → Position Embedding

Word embeddings carry meaning but not order — the word "cat" gets the same vector whether it's the first or the fifth token. Position embeddings fix this by assigning a distinct vector to each slot in the sentence. We assume sentences of up to 21 tokens (positions 0–20). Here we use the classic sinusoidal formula:  PE(p) = [sin(p·ω), cos(p·ω)], so every position lands on the unit circle.

Word Embedding + Position Embedding = Final Embedding

The transformer's actual input to the first layer is the sum of the word embedding and the position embedding. Same word at a different position → different final vector. The plot shows all three vectors for each token together; a dashed line connects each word embedding to its final embedding (the displacement is exactly the position embedding).
